# Geo_Tracker_app

Geo_Tracker_app is an Android application that allows users to track their current location, save important places, receive geofence alerts, and navigate to saved locations using Google Maps.

## Features

- Real-time GPS location tracking
- Add and save custom locations
- Local storage using Room database
- Geofencing with entry alerts
- Notifications on entering saved locations
- Navigation using Google Maps
- Clean Material Design UI

## Technologies Used

- Java (Android)
- Room Database
- FusedLocationProviderClient
- GeofencingClient
- NotificationManager
- Google Maps Intents

## Minimum Requirements

- Android API 26+
- Location services enabled

## How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator or device
5. Grant required permissions

## Permissions Required

- ACCESS_FINE_LOCATION
- ACCESS_COARSE_LOCATION
- ACCESS_BACKGROUND_LOCATION
- POST_NOTIFICATIONS
